# jQuery Bootstrap Token Chooser Plugin

## Example Usage

### HTML

```html
<div id="chooser"></div>
```

### jQuery

Use the plugin as follows:

```js
$('#chooser').chooser({
    data: ['option 1', 'option 2', 'option 3']
});
```
